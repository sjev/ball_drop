# ball_drop
Build an arduino-based robot with a gripper, used for fun demonstrations


## Pinout

| Pin           |  Function |
| ------------- | ------------- |
| 2  | input&interrupt, encoder  |
| 4  | output, solenoid actuator  |
| 9  | output, red led |
| 13 | output, onboard led for debugging |
